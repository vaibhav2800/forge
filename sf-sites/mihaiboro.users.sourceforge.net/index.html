Projects

JEncConv and PyEncConv Encoding Converters

Text file encoding converters. Both perform essentialy the same function, and can be used from a GUI or the command line. In some special cases you might prefer one over the other. See their websites JEncConv and PyEncConv for details.

DiceLottery

A cross-platform Java program which maps (random) events from a source (an available generator) to a desired destination. For example, you can throw a pair of dice (several times) and map the result to lottery numbers to bet on.

TTCN-3 Framework for Java Unit Testing

A research project (compiled as an Eclipse plugin for Testing Technologies' TTworkbench) allowing TTCN-3 programmers to make Java calls, and in particular to perform Java unit tests.

SimpleSwing

A Java library used by some of my other projects. Offers support for easily building a Swing GUI.
