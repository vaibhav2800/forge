JFileConv

JFileConv is a text file encoding converter. It reads a text file using an encoding (e.g. ISO-8859-2) and writes the characters to a file using another encoding (e.g. UTF-8).

It supports plugins (for processing the text) and has a preview function which allows the user to see how the file is decoded with a particular charset.

Download

Download the latest version from sourceforge. The program is distributed as a .zip archive containing a (cross-platform) .jar file and a few other required files. After unzipping, you may run the program by double-clicking the .jar or by typing java -jar jfileconv.jar in a terminal. You need Java installed.
